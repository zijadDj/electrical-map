## Usage

1. Open the application in your web browser https://electrical-map-wv2hp.ondigitalocean.app/map
2. Use the sidebar to filter and search for electrical components
3. Click on map markers to view detailed information
4. Use the search functionality to find specific components
5. Enable location services to see your position on the map
